## koa
:::tip 中间件

- koa-router 路由
- cors 跨域
- koa-body 参数解析
- mysql2/sequelize 数据库
- bcryptjs 密码加密
- jsonwebtoken 用户Token

:::

:::info 项目结构

- controller 控制层
- sql 数据库相关
- middleware 中间件
- model 模型
- router 路由
- service 服务层
:::
