## SOME

### == 和 ===

- `===` 严格相等（三个等号），不进行类型转换
- `==` 宽松相等（两个等号），执行类型转换
- `Object.is()` Object.is(nan2, NaN) 判断某值是否为 NaN

**在大多数情况下，不建议使用宽松相等。使用严格相等进行比较的结果更容易预测，并且由于缺少类型强制转换可以更快地执行。**
